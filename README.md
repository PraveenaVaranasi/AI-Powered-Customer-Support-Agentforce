# AI-Powered Customer Support Automation using Salesforce Agentforce
## Overview
This project demonstrates how Salesforce Agentforce can be used to automate customer support
operations by handling queries, creating cases, and routing them efficiently using AI-driven workflows.
## Technologies Used
- Salesforce Agentforce
- Salesforce Flow Builder
- Case Management
- Trailhead (SB_TCS VIP 2025)
## Key Features
- AI-powered customer query handling
- Automatic case creation and assignment
- Workflow automation using Salesforce Flows
- Improved customer support efficiency
## Outcome
This project showcases practical knowledge of Salesforce Agentforce and AI-based automation in CRM systems.
